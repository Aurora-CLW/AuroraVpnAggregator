|---------|----------------------|

| 复杂度等级 | 中等 (26/50) |

---

## 核心目标

构建一个全自动化的 VPN 订阅聚合系统，实现：

1. **多源聚合** - 支持 GitHub 订阅、本地文件、Telegram 频道
2. **智能测试** - 自动检测节点有效性，过滤失效节点
3. **多格式输出** - 支持 Clash、V2Ray、Sing-box 三种格式
4. **自动更新** - GitHub Actions 定时执行，无需人工干预

---

## 功能清单

| 序号 | 功能模块 | 功能描述 | 优先级 |
|------|---------|---------|--------|
| 1 | 订阅源管理 | 支持 GitHub URL、本地文件、Telegram 频道三种订阅源 | P0 |
| 2 | 节点解析 | 解析 SS/VMess/VLess/Trojan/Hysteria2 等多种协议 | P0 |
| 3 | 节点测试 | TCP 可达性测试 + HTTP 代理连通性测试 | P0 |
| 4 | 节点去重 | 基于指纹的去重算法，避免重复节点 | P0 |
| 5 | 订阅生成 | 生成 Clash YAML、V2Ray Base64、Sing-box JSON | P0 |
| 6 | 自动部署 | GitHub Actions 定时执行，GitHub Pages 托管 | P0 |
| 7 | 地理识别 | 自动识别节点所属国家/地区 | P1 |
| 8 | 统计报告 | 生成节点统计信息，包含分布、有效性等 | P1 |
| 9 | 通知推送 | 支持更新完成后的通知推送 | P2 |

## 技术架构

```
┌─────────────────────────────────────────────────────────────────┐
│                      GitHub Actions (每 6 小时)                 │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  订阅源采集层                                                   │
│  ├── GitHub Handler    - 抓取 GitHub 订阅                      │
│  ├── Local Handler     - 读取本地文件                          │
│  └── Telegram Handler  - 解析 Telegram 频道消息                │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  节点处理层                                                     │
│  ├── Parser          - 多格式解析                              │
│  ├── Deduplicator    - 指纹去重                                │
│  └── GeoIP           - 地理识别                                │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  节点测试层                                                     │
│  ├── TCP Tester      - 端口可达性测试 (并发 100)               │
│  └── HTTP Tester     - 代理连通性测试 (并发 20)                │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  订阅生成层                                                     │
│  ├── Clash Generator   - 生成 Clash YAML                      │
│  ├── V2Ray Generator   - 生成 V2Ray Base64                    │
│  └── Singbox Generator - 生成 Sing-box JSON                   │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  发布托管层                                                     │
│  ├── GitHub Pages  - 静态文件托管 + CDN                        │
│  └── GitHub Release - 历史版本存档                             │
└─────────────────────────────────────────────────────────────────┘
```
